# Brightly — Employee Management System

A self-contained, single-page HR dashboard for managing employee records. Open `roster.html` in any modern browser — no install or server required.

## Features

- **Dashboard** — KPI cards, work performance chart, attendance gauge, and employee preview table
- **Employee management** — Add, edit, delete, search, and filter employees
- **Departments** — Browse teams grouped by department
- **Dark UI** — Modern "Brightly" dashboard design with sidebar navigation
- **Persistent storage** — Data saved locally in your browser (localStorage)

## Quick Start

1. Open [`roster.html`](roster.html) in Chrome, Edge, or Firefox
2. Use **Add New** to create employees
3. Navigate via the sidebar: Dashboard, Employee, Departments, and more

## Keyboard Shortcut

Press **⌘K** (Mac) or **Ctrl+K** (Windows) to focus the global search bar.
